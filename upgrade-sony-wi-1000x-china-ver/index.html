强制更新国行 Sony WI-1000X 2.0.1固件

2018-11-06

似乎是因为支持 Google Assistant 的原因，国行的 Sony 降噪耳机都没有 2.0.1 的固件更新。今天 @xream 给我发了个链接 这篇文章有魔改 Sony Headphone Connect apk 来强制更新 WH-1000XM2 的方法，那么同理，WI-1000X 也能这么改。

方法

具体方法 Howard 的博客里写了就不重复了，主要是找对应的产品ID。
用 Packet Capture 抓包获得请求http://info.update.sony.net/HP001/MDRID285302/info/info.xml ，那么国行 WI-1000X 的内部产品ID就是285302了。

依此类推得出一个表：

型号 版本 产品ID 请求URL
WI-1000X 国际版/英文 285300 URL
WI-1000X 日本版/日文? 285301 URL
WI-1000X 中国版/中文 285302 URL
WH-1000XM2 国际版/英文 289200 URL
WH-1000XM2 日本版/日文? 289201 URL
WH-1000XM2 中国版/中文 289202 URL
WH-1000XM3 国际版/英文 291600 URL
WH-1000XM3 日本版/日文? 291601 URL
WH-1000XM3 中国版/中文 291602 URL

果然每个中国版的 URL 都是404。之后就按照文章里的方法反编译固定写对应ID再编译签名就行了。

小插曲

我一开始使用的最新版 Sony Headphone Connect 4.0.1，反编译发现相关代码都被索尼改完了所以还是用的 3.2.0 版

刷机

删掉官方版 Sony Headphone Connect 安装修改版，连接耳机，就能看见更新提示了，下载飞快但是蓝牙传输到耳机花了半小时，更新完自动重启就可以正常使用了。再装回原版 Sony Headphone Connect 也不会影响功能，而且终于也看到了新的 Google Assistant 选项了。
